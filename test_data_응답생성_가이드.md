## 응답 데이터 생성 시 참고사항

### RESPONSES 테이블 구조
- `ID`: AUTO_INCREMENT
- `RESPONSE_SESSION_ID`: UUID 형식 (같은 세션의 응답은 동일한 값)
- `QUESTION_ID`: 질문 ID (1-29)
- `QUESTION_TYPE`: 'MULTIPLE_CHOICE' 또는 'SUBJECTIVE'
- `OPTION_ID`: 객관식 질문인 경우 선택한 옵션 ID (NULL 가능)
- `SUBJECTIVE_CONTENT`: 주관식 질문인 경우 답변 내용 (NULL 가능)
- `SCALE_VALUE`: 척도형 질문인 경우 값 (현재 미사용, NULL)
- `CREATED_AT`: 응답 생성 시간

### 응답 생성 규칙
1. **같은 설문의 응답은 같은 RESPONSE_SESSION_ID를 사용**
   - 예: Survey 1에 응답하면 Question 1, 2 모두 같은 RESPONSE_SESSION_ID

2. **객관식 질문 (MULTIPLE_CHOICE)**
   - `OPTION_ID`: 해당 질문의 옵션 ID 중 하나 선택
   - `SUBJECTIVE_CONTENT`: NULL
   - `QUESTION_TYPE`: 'MULTIPLE_CHOICE'

3. **주관식 질문 (SUBJECTIVE)**
   - `OPTION_ID`: NULL
   - `SUBJECTIVE_CONTENT`: 답변 텍스트
   - `QUESTION_TYPE`: 'SUBJECTIVE'
