## 🛠  사용기술

### **Frontend**
- Vue3, Typescript
- [thinkfast-vue3 리포지토리로 이동하기](https://github.com/kwakchaewon/thinkfast-vue3)

### **Backend**
- Spring boot, Spring Security, Redis, Web Socket

### **Database**
- MariaDB + Redis

### **AI / LLM**
- Python 기반 LangChain, Hugging Face Transformers
- Vector DB (FAISS)

### **Infra / Deployment**
- AWS EC2 & RDS
- Docker & Jenkins 기반 CI/CD
- HTTPS only Cookie
- Redis Pub/Sub
